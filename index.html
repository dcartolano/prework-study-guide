Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content Shown to the user.
  • In semantic HTML elements, the name of the element describes its content and the role it plays on a webpage.
  • Elements that are nested inside other elements are called child elements.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the declarations apply.
  • A declaration contains two components: the property we want to apply and the value of that property.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls down the latest main branch
  • git push origin branch-name: pushes our changes to the remote branch
